# s3

[Amazon S3](https://aws.amazon.com/s3/)

## Config

### Servicer

| Name         | Required | Comments                          |
| ------------ | -------- | --------------------------------- |
| `credential` | Y        | support `hmac` and `env` protocol |

### Storager

| Name       | Required | Comments        |
| ---------- | -------- | --------------- |
| `name`     | Y        | bucket name     |
| `work_dir` | N        | work dir        |
| `location` | Y        | bucket location |

## Example

Init servicer

```yaml
credential: hmac:<account_name>:<account_key>
```

Init storager

```yaml
credential: hmac:<account_name>:<account_key>
name: <container_name>
work_dir: /<work_dir>
location: <bucket_location>
```
